# Competitive Research Dimensions

Research dimensions split into two layers:
- **Fixed (17)** — universal across all industries, always included
- **Variable (8-15)** — industry-specific, selected based on the client's sector

## Fixed Dimensions (always included)

### A. Identification (3)

| # | Dimension | Description |
|---|-----------|-------------|
| 1 | **competitor_name** | Full name (in the market language) |
| 2 | **competitor_type** | Category within the industry (e.g., "independent", "subsidiary", "marketplace-native") |
| 3 | **parent_company** | Parent group/holding, or "Independent" |

### B. Brand & Positioning (5)

| # | Dimension | Description |
|---|-----------|-------------|
| 4 | **brand_essence** | Core brand promise in 1-2 sentences. Quote actual tagline/slogan if available (in «кавычках»). |
| 5 | **brand_positioning** | Market positioning: mass-market, premium, niche, tech-forward, expert, accessible, etc. |
| 6 | **brand_archetype** | Dominant archetype (Hero, Sage, Creator, Magician, Explorer, Ruler, Caregiver, Regular Guy, Rebel, etc.) with 1-sentence justification. |
| 7 | **brand_image** | Visual identity: logo, primary colors (hex if possible), design language, aesthetic. |
| 8 | **tone_of_voice** | Communication tone: formal/informal, expert/accessible, aggressive/soft. Include 1-2 verbatim examples. |

### C. Communication & Channels (4)

| # | Dimension | Description |
|---|-----------|-------------|
| 9 | **communication_channels** | All known channels: TV, digital, social, outdoor, content, events, sponsorships, influencer. |
| 10 | **social_media_presence** | Key accounts with follower counts. Cover: Telegram, VK, YouTube, Instagram, TikTok, Дзен (RU) or LinkedIn, X, Facebook (int'l). |
| 11 | **content_and_education** | Owned media: blogs, podcasts, newsletters, webinars, courses, media brands. Name specific products. |
| 12 | **key_messages** | 3-5 recurring themes in recent marketing. What do they repeatedly communicate? |

### H. Market Position (4)

| # | Dimension | Description |
|---|-----------|-------------|
| 13 | **market_share_data** | Revenue, client count, industry rankings, trading volumes, production volumes — whatever applies. Use most recent data, flag vintage. |
| 14 | **target_audience** | Primary target: segments, demographics, firmographics. Describe their ideal client/customer. |
| 15 | **competitive_advantages** | 3-5 key advantages vs. other players in the market. |
| 16 | **competitive_weaknesses** | 3-5 known weaknesses from public reviews, research, market perception. |

### I. Source Attribution (1)

| # | Dimension | Description |
|---|-----------|-------------|
| 17 | **sources** | Semicolon-separated list of key URLs/sources. Minimum 3 per competitor. |

## Composing the Final Dimension Set

### Step 1: Start with Fixed 17

Always include dimensions 1-17 (groups A, B, C, H, I).

### Step 2: Auto-detect industry from brief

Read the client brief and identify the closest industry preset. Signals:
- Products/services described
- Competitors mentioned
- Market terminology used
- Regulatory context

### Step 3: Propose variable dimensions

Present the matched preset to the user:

```
Based on the brief, I've identified this as a **[Industry]** project.

**Fixed dimensions (always included):** 17
- A. Identification (3), B. Brand & Positioning (5), C. Communication (4), H. Market Position (4), I. Sources (1)

**Suggested variable dimensions from [Industry] preset:** [N]
- V1. [dimension] — [one-line description]
- V2. [dimension] — ...
...

**Total: [17 + N] dimensions per competitor**

Would you like to:
- Add dimensions from another preset?
- Add custom dimensions specific to this project?
- Remove any suggested dimensions?
```

### Step 4: Let user modify

Accept user additions, removals, or custom dimensions. Custom dimensions should follow the format:
```
V[N]. **dimension_name** — Description of what to research and how to present the finding.
```

### Step 5: Number the final set

After confirmation, number all dimensions sequentially:
- Fixed 1-17
- Variable 18-N (renumbered from V1, V2, etc.)

This final numbered set goes into the research template.

## Suggesting Additional Dimensions

After presenting the preset, proactively suggest dimensions the user might not have considered. Base suggestions on:

1. **Cross-industry patterns** — Dimensions from other presets that might apply:
   - FMCG project with an e-commerce arm? Suggest `ux_and_conversion`, `logistics_and_delivery`
   - Tech company selling to enterprise? Suggest `key_accounts`, `methodology` from Services
   - Finance project with a mobile-first competitor? Suggest deeper `ai_capabilities`

2. **Strategic context** — Based on the brief's stated goals:
   - Repositioning toward premium? Suggest premium-specific dimensions (premium offering, affluent communication)
   - International expansion? Suggest `export_markets`, `localization`, `regional_coverage`
   - Digital transformation? Suggest `ai_capabilities`, `digital_maturity`, `technology_stack`

3. **Competitive asymmetry** — When competitors span different types:
   - Mix of traditional and digital-native? Add `digital_transformation`
   - Mix of local and international? Add `geographic_reach`
   - Mix of B2B and B2C? Add both dimension sets

Present suggestions as: "Based on the strategic context, you might also want to add: ..."
