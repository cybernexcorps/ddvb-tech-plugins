# Phase 2: Strategic Plan

You are starting the **Plan** phase. The Spec phase must be completed first (`analysis/spec_summary.md` should exist).

## Prerequisites

- Read `analysis/spec_summary.md` to understand the brief
- Confirm which hypothesis the user has chosen
- Have all extracted data available

## Deliverables (in order)

### Block 1: Analytical

1. **SWOT analysis** — Create `analysis/swot_competitive.md`
   - 7-9 factors per quadrant, each with source attribution
   - Link strengths/weaknesses to opportunities/threats (SO/WT strategies)

2. **Competitive landscape** — Include in the same file
   - Positioning map axes (define the 2 dimensions)
   - Detailed competitor matrix (awareness, archetype, key factor, threat level)
   - Competitive differentiation table for chosen hypothesis

### Block 2: Strategic

3. **Brand platform** — Create `analysis/brand_platform.md`
   - Mission statement (with decomposition and competitor contrast)
   - Client value proposition (formula + pyramid)
   - 3-4 brand values as active principles (each with archetype link, rational/emotional manifestation, communication marker, antithesis)
   - Values hierarchy (foundation → promise)
   - Archetype formula with character traits
   - Brand voice examples (old vs new, across 4+ situations)
   - Big Idea (generative formula: Big Idea + object = message)
   - 3-5 alternative Big Idea formulations for discussion

4. **Brand architecture** — Include in brand_platform.md
   - Architecture model (endorsed monolith, house of brands, etc.)
   - Sub-brand mapping to values and Big Idea variations
   - Media satellite roles

### Block 3: Communication

5. **Communication platform** — Create `analysis/communication_platform.md`
   - Communication territory and tone of voice table
   - Key communication idea + brand manifesto
   - 3 key messages (each tied to a value, with rational + emotional levels, proof points, target barrier, format)
   - 5 slogans (1 main tagline + 4 tactical, with usage contexts)
   - Channel × message matrix
   - Content pillars with percentage allocation
   - Funnel mapping (awareness → consideration → conversion)

### Block 4: Visual Concept

6. **Visual identity** — Create `analysis/visual_identity.md`
   - Visual concept name and metaphor
   - Color system (primary, secondary, neutral) with hex codes, roles, and rules
   - Typography system (font families, hierarchy table, rules)
   - Graphic device (describe 3-4 variants with characteristics)
   - Photo style guidelines
   - Layout concepts for digital banner + outdoor ad
   - Design tokens (spacing, radius, shadows)

## Important

- Each analysis file should be self-contained and presentation-ready
- Use tables for structured comparisons
- Include source references where possible
- Ask the user to confirm Big Idea before proceeding to review
